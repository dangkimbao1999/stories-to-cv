# Plan Writing

Rules:

- write plans in English
- read related plans before creating a new one
- use `docs/plans/TEMPLATE.md`
- plans should explain the `how`, not re-explain the whole product vision
- reference prior plans when extending or replacing earlier work

Primary reference:

- `docs/guides/plan-writing-for-codex.en.md`
