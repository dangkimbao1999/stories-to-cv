# Codex Operating Model

This folder contains the Codex-native operating model for this repository.

- `agents/`: focused role descriptions for the current phase
- `rules/`: mandatory repo rules for TDD, git/worktrees, and local development
- `worktrees/`: default location for auxiliary worktrees
- `logs/`: local log files produced by helper scripts
- `environments.example.json`: environment command templates for local logs and verification
- `.agents/skills/`: active repo-local skills for TDD, git workflow, local startup, and logs

The goal is to keep the agent context lean while preserving the habits needed for the current development phase.
